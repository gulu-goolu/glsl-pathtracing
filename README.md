# glsl-raytracing

玩票性质的小项目，使用 glsl 来实现光线跟踪算法。

## build

首先在你的计算机上安装下列软件：

- Vulkan SDK
- CMake
- Git
- Visual Studio

```bash
# 拉取代码
git clone -b master https://github.com/murmur-wheel/glsl-raytracing.git
cd glsl-raytracing
mkdir cmake-build && cd cmake-build
cmake ..
```

然后使用 visual studio 打开你的项目
